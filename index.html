Nuestro Proyecto

El proyecto AirGuard tiene como objetivo mejorar la calidad ambiental al monitorear aspectos clave como la calidad del aire, la radiación UV, la humedad y la temperatura. Mediante el uso de sensores avanzados, buscamos recopilar datos en tiempo real sobre estos factores, lo que nos permitirá identificar patrones y variaciones en el ambiente. A través de este monitoreo constante, nuestro objetivo es proporcionar a las personas herramientas para tomar decisiones informadas sobre su salud y seguridad, además de contribuir a un entorno más saludable para todos.

Termógrafo

Medición de la Calidad del Aire

Paso 1: Calibración
Calibrar el sensor de calidad del aire para asegurar lecturas precisas utilizando un gas de referencia o un índice de calidad del aire conocido.
Paso 2: Recolección de Datos
Colocar el sensor de calidad del aire en diferentes ubicaciones para recolectar datos en tiempo real sobre contaminantes, materia particulada y concentraciones de gases.
Paso 3: Análisis de Datos
Analizar los datos recolectados utilizando herramientas de software o algoritmos para evaluar los niveles de calidad del aire y compararlos con las normas de seguridad.

Medición de la Radiación UV

Paso 1: Instalación del Sensor UV
Instalar el sensor de radiación UV en un área con exposición directa a la luz solar o fuentes de radiación UV.
Paso 2: Monitoreo Continuo
Monitorear de manera continua los niveles de radiación UV en diferentes momentos del día para comprender las fluctuaciones según la intensidad de la luz solar.
Paso 3: Registro de Datos
Registrar y almacenar los datos de radiación UV para su posterior análisis y comparación con las pautas de salud sobre exposición a la radiación UV.

Medición de la Humedad

Paso 1: Calibración del Sensor
Calibrar el sensor de humedad para asegurar lecturas precisas comparándolo con una fuente de humedad conocida.
Paso 2: Colocación de los Sensores
Instalar el sensor de humedad en diferentes ambientes para medir los niveles de humedad en condiciones variadas (por ejemplo, interiores y exteriores).
Paso 3: Registro y Análisis de Datos
Registrar los datos de humedad y analizar las tendencias para identificar patrones o cambios a lo largo del tiempo.

Medición de la Temperatura

Paso 1: Calibración del Sensor
Calibrar el sensor de temperatura comparándolo con un termómetro estándar o un control de temperatura.
Paso 2: Colocación del Sensor
Colocar el sensor de temperatura en diferentes ambientes para monitorear los cambios de temperatura a lo largo del día.
Paso 3: Registro de Datos
Registrar los datos de temperatura para su análisis y uso en la creación de un perfil térmico del área monitoreada.
